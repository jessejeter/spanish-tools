# Spanish Tools

A collection of tools to help with learning Spanish.

## Features

- Vocabulary practice
- Verb conjugation drills
- Flashcard system
- Progress tracking

## Getting Started

```bash
# Clone the repository
git clone https://github.com/jessejeter/spanish-tools.git
cd spanish-tools

# Install dependencies
npm install

# Run the application
npm start
```
